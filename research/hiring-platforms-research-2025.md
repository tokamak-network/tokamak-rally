# 채용 플랫폼 & ATS 리서치 2025-2026

> Tokamak Hiring System 개선을 위한 경쟁 분석 및 벤치마킹 리포트

---

## 1. 주요 ATS / 채용 플랫폼

### Greenhouse
- **핵심 기능**: 구조화된 인터뷰 킷 & 스코어카드, 소싱 CRM, AI 리포트 필터, 소싱 자동화, 후보자 셀프 스케줄링, 이력서 익명화(DE&I), take-home 테스트 스케줄링
- **가격**: 비공개 (Core/Plus/Pro 3단계). 일반적으로 연 $6,000~$30,000+ (직원 수 기준)
- **강점**: 구조화 채용 프로세스의 표준. 500+ 통합. Scorecard 시스템이 매우 체계적
- **API/통합**: 풍부한 API, HackerRank/CodeSignal 직접 연동, Slack, LinkedIn RSC
- **배울 점**: **구조화된 평가 스코어카드** → Tokamak Track B의 리뷰어 평가 시스템에 적용 가능. 평가 항목별 점수 + 종합 의견 구조

### Lever (현재 Employ 산하)
- **핵심 기능**: ATS + CRM 통합, 비주얼 파이프라인, 다이버시티 서베이, 자동 아웃리치 시퀀스
- **가격**: 비공개 (일반적 $3,000~$15,000/년)
- **강점**: CRM이 ATS에 네이티브 통합. 소싱과 파이프라인 관리가 하나의 플로우
- **배울 점**: **CRM + ATS 통합 모델** → 후보자와의 장기 관계 관리. Track A/B 지원자를 talent pool로 관리

### Ashby
- **핵심 기능**: 올인원 (소싱, 아웃리치, ATS, 스케줄링, 애널리틱스), 강력한 리포팅
- **가격**: Foundations $400/월 (100명 이하), Plus/Enterprise는 맞춤 견적
- **강점**: 스타트업~중견기업 최적화. 실시간 분석. 빠른 설정
- **배울 점**: **올인원 접근** 및 **데이터 기반 채용 분석**. 채용 퍼널 단계별 전환율 추적

### Workable
- **핵심 기능**: AI 후보자 추천, 200+ 잡보드 자동 포스팅, 비디오 인터뷰, 레퍼런스 체크
- **가격**: Starter $189/월, Standard ~$313/월, Premier 맞춤
- **강점**: 중소기업에 가성비 좋음. AI 추천 기능
- **배울 점**: **잡보드 자동 배포** → 블록체인 잡보드 다중 게시 자동화

### BambooHR
- **핵심 기능**: HR 종합 솔루션 (ATS는 부가 기능), 온보딩, 급여, 성과관리
- **가격**: ~$6/직원/월 (Core), ~$8/직원/월 (Pro)
- **강점**: HR 전체 라이프사이클 관리
- **배울 점**: 제한적 (ATS 특화가 아님). 다만 **온보딩 자동화** 참고 가치

## 2. AI 기반 소싱 도구

### hireEZ (구 Hiretual)
- **핵심 기능**: 오픈 웹 소싱 (LinkedIn, GitHub, Stack Overflow 등 크롤링), 에이전틱 AI, CRM, 자동 아웃리치, 내부 이동성, 40+ ATS 연동
- **소싱 방식**: 800M+ 프로필 인덱싱. Boolean 검색 + AI 매칭. GitHub 포함한 다중 소스
- **가격**: ~$169/사용자/월 (Professional), Enterprise 맞춤
- **배울 점**: **오픈 웹 소싱** 개념 → GitHub 프로필 + 온체인 활동 자동 크롤링으로 확장 가능

### Gem
- **핵심 기능**: 소싱 CRM, 이메일 시퀀스 자동화, 파이프라인 분석, LinkedIn 통합
- **소싱 방식**: LinkedIn + 이메일 통합. 시퀀스 기반 아웃리치
- **가격**: 맞춤 견적 (~$5,000+/년)
- **배울 점**: **자동 아웃리치 시퀀스** → Track B 관심 개발자에게 단계적 접근 메시지

### SeekOut
- **핵심 기능**: AI 소싱, 다이버시티 필터, GitHub/특허 검색, 기술 프로필 분석
- **소싱 방식**: LinkedIn + GitHub + 특허 + 학술 데이터. 기술 스택별 필터
- **가격**: ~$10,000+/년
- **배울 점**: **GitHub 프로필 깊이 분석** → 기여 패턴, 사용 언어, 프로젝트 복잡도 분석

### Fetcher
- **핵심 기능**: AI + 인간 큐레이터 결합 소싱, 자동 아웃리치
- **소싱 방식**: AI가 후보자 리스트 생성 → 큐레이터가 검증 → 자동 이메일
- **가격**: ~$249~$549/월
- **배울 점**: **AI + 인간 큐레이터 하이브리드** 모델 → 리뷰어 매칭에 적용 가능

### Findem
- **핵심 기능**: "속성 기반" 소싱, 3D 프로필 (다중 데이터 소스 통합), 예측 분석
- **소싱 방식**: 공개 데이터 750+ 속성으로 프로필 생성. "다음에 이직할 확률" 예측
- **가격**: 맞춤 견적 (보통 $10,000+/년)
- **배울 점**: **속성 기반 매칭** → 기술 스택, 컨트리뷰션 유형, 온체인 활동 등 다차원 속성으로 후보자 매칭

## 3. 개발자 특화 채용 플랫폼

### Triplebyte (현재 Karat에 흡수)
- **핵심 기능**: 기술 역량 퀴즈 + 기술 인터뷰로 선별 → 기업 매칭
- **평가 방식**: 온라인 퀴즈 → 2시간 기술 인터뷰 (코딩 + 시스템 설계)
- **현황**: 2023년 Karat에 인수. 독립 플랫폼으로서는 사실상 종료
- **배울 점**: **코딩 능력 기반 사전 선별** 후 기업 매칭 모델. 이력서 없이 실력으로 평가하는 선구자

### Hired
- **핵심 기능**: 역채용 마켓플레이스. 개발자가 프로필 등록 → 기업이 제안
- **평가 방식**: 기술 평가 (skills assessment) + 급여 기대치 매칭
- **가격**: 채용 성공 시 연봉 15% 수수료
- **배울 점**: **역채용 모델** (후보자가 등록 → 기업이 찾아감). Track B와 유사한 개념

### Turing
- **핵심 기능**: 글로벌 원격 개발자 풀. AI 기반 매칭. 벳팅 프로세스
- **평가 방식**: AI 코딩 테스트 + 기술 인터뷰 + 프로젝트 시뮬레이션
- **현황**: AI 데이터 라벨링/훈련 사업으로 피벗 중
- **배울 점**: **다단계 벳팅** (자동 → 수동). 글로벌 원격 인력 매칭

### Andela
- **핵심 기능**: 아프리카/글로벌 개발자 커뮤니티. 기업과 장기 계약 매칭
- **평가 방식**: 기술 평가 + 소프트스킬 + 코드 리뷰
- **가격**: 시급 기반 ($40~$80+/시간)
- **배울 점**: **커뮤니티 기반 인재 풀** 구축 모델

### TopCoder (Topcoder)
- **핵심 기능**: 경쟁 프로그래밍 기반 인재 검증. 크라우드소싱 개발
- **평가 방식**: 대회 성적 = 실력 증명. 알고리즘, UI/UX, QA 등 분야별 대회
- **배울 점**: **대회/챌린지 기반 실력 검증** → Track B의 deliverable 평가와 유사한 철학

## 4. 코드/스킬 평가 플랫폼

### HackerRank
- **핵심 기능**: 코딩 테스트 + 라이브 인터뷰, AI 프록토링, 표절 감지, 7,500+ 문제
- **가격**: Starter $165/월, Pro $375/월, Enterprise 맞춤
- **통합**: Greenhouse, Lever, Ashby ATS 연동
- **AI 기능**: AI Proctoring, Identity Verification, Scorecard Assist
- **배울 점**: **자동화된 코드 평가 + 표절 감지** → deliverable 평가 시 원본성 검증에 참고

### CodeSignal
- **핵심 기능**: 기술 역량 평가 (Pre-Screen, Interview), AI 기반 스코어링, 학습 플랫폼
- **가격**: 맞춤 견적 (일반적 $10,000+/년)
- **AI 기능**: Coding Score (표준화 점수), AI 기반 평가 자동화
- **배울 점**: **표준화된 기술 점수** (Coding Score) 개념 → deliverable 품질의 표준화 점수 부여

### LeetCode Enterprise
- **핵심 기능**: 면접 문제 풀, 기업 맞춤 문제 세트, 온라인 판단
- **가격**: Teams ~$299/년/사용자
- **배울 점**: 알고리즘 중심이라 Tokamak과는 방향이 다름. 다만 **문제 뱅크 관리** 시스템 참고

### Codility
- **핵심 기능**: 실시간 코딩 테스트, 맞춤 태스크, 자동 채점, 표절 감지
- **가격**: Starter ~$500/월
- **배울 점**: **커스텀 태스크 생성** → Track B에서 프로젝트별 맞춤 과제 생성

## 5. 블록체인/Web3 특화 플랫폼

### CryptoJobsList
- **특징**: Web3 전문 잡보드. 단순 게시판 모델. 기업이 포스팅 비용 지불
- **가격**: 기본 $199/포스트, Featured $349/포스트
- **한계**: 기본 잡보드. 평가/매칭 기능 없음

### Web3.career
- **특징**: Web3 잡 애그리게이터. 블록체인, 스마트컨트랙트, DeFi 등 카테고리별 분류
- **가격**: 무료 게시 + 프리미엄 옵션
- **한계**: 역시 단순 잡보드. ATS 기능 없음

### Cryptocurrency Jobs
- **특징**: 크립토 전용 잡보드. 주요 프로토콜/거래소 채용 공고 집중
- **한계**: 잡보드만 제공

### Gitcoin
- **특징**: 바운티/그랜트 기반 기여자 발굴. 실제 코드 기여로 실력 검증
- **가격**: 플랫폼 수수료 (그랜트 라운드별 상이)
- **핵심 차별점**: **실제 기여 = 실력 증명** → Track B와 가장 유사한 철학!
- **배울 점**: 바운티 시스템, 온체인 기여 이력, Gitcoin Passport (신원 검증)

### 기타 Web3 채용 채널
- **Pallet** (커뮤니티 기반 리크루팅)
- **Station** (Web3 인재 플랫폼)
- **Dework** (Web3 태스크/바운티 플랫폼 - Gitcoin과 유사)
- **Layer3** (퀘스트 기반 온보딩 → 채용으로 연결)

## 6. 혁신적 접근 방식

### 포트폴리오/프로젝트 기반 채용
- **Peerlist**: 개발자 포트폴리오 + 프로젝트 쇼케이스 기반 채용
- **Polywork**: 프로젝트/성과 중심 프로필 (이력서 대체)
- **ReadMe/DevPost**: 해커톤 프로젝트를 포트폴리오로 활용
- **배울 점**: Track B의 deliverable이 곧 포트폴리오가 되는 구조

### GitHub 프로필 분석 AI 도구
- **GitStar Ranking / OSS Insight**: GitHub 활동 분석
- **SeekOut**: GitHub 프로필 심층 분석 (언어, 기여 빈도, 프로젝트 영향력)
- **SourceGraph**: 코드 검색 기반 인재 발굴
- **Commit.dev** (폐업): GitHub 분석 기반 매칭을 시도했으나 실패
- **배울 점**: GitHub 분석 단독으로는 불충분. **실제 과제 수행 + GitHub 이력 결합**이 더 강력

### 내부 리뷰어 자동 매칭
- **Greenhouse**: 인터뷰어 역량/선호도 기반 자동 스케줄링
- **Karat**: 전문 인터뷰어 네트워크 (외부 전문가가 기술 면접 대행)
- **배울 점**: **리뷰어 전문성 태그 + 부하 분산 알고리즘** → Track B 과제 리뷰어 자동 배정

## 7. Tokamak Hiring System 권고사항

### 🔑 우리의 고유 차별점
1. **Track B (Deliverable 기반)**: 이력서 없이 실제 결과물로만 평가 → 업계에서 Gitcoin 바운티만이 유사, 하지만 "채용 프로세스"로서는 거의 유일
2. **블록체인 네이티브**: 온체인 기여 이력 + 평가 결과의 투명성
3. **리뷰어 매칭**: 내부 팀원이 직접 deliverable을 리뷰 → 팀 핏 자연스럽게 검증
4. **No Resume 철학**: Triplebyte가 시도했으나 실패, 우리는 더 구체적인 (프로젝트 기반) 접근

### 📋 기존 플랫폼에서 채택할 기능 (우선순위순)

#### P0 (즉시 구현)
| 기능 | 참고 플랫폼 | 이유 |
|------|-----------|------|
| **구조화된 평가 스코어카드** | Greenhouse | 리뷰어의 평가를 항목별로 표준화 (코드 품질, 아키텍처, 완성도 등) |
| **자동 아웃리치 시퀀스** | Gem, hireEZ | Track B 관심 개발자에게 단계적 접근 이메일/DM 자동화 |
| **표절/원본성 검증** | HackerRank, Codility | Deliverable이 본인 작성인지 자동 검증 (코드 유사도, AI 생성 감지) |

#### P1 (3개월 내)
| 기능 | 참고 플랫폼 | 이유 |
|------|-----------|------|
| **GitHub 프로필 자동 분석** | SeekOut, hireEZ | 지원자의 GitHub 활동 자동 크롤링 → 기술 스택/기여도 프로필 생성 |
| **리뷰어 자동 매칭** | Greenhouse, Karat | 과제 기술 스택 ↔ 리뷰어 전문성 매칭 + 부하 분산 |
| **파이프라인 분석 대시보드** | Ashby | Track A/B 전환율, 단계별 이탈률, 평균 처리 시간 추적 |
| **다중 Web3 잡보드 동시 게시** | Workable | CryptoJobsList, Web3.career, Cryptocurrency Jobs 동시 포스팅 |

#### P2 (6개월 내)
| 기능 | 참고 플랫폼 | 이유 |
|------|-----------|------|
| **온체인 활동 분석** | Gitcoin Passport | 지갑 주소 → DeFi/DAO 기여 이력 분석 → 블록체인 역량 자동 프로파일링 |
| **AI 후보자 스코어링** | CodeSignal, Findem | Deliverable 자동 1차 평가 (코드 품질, 테스트 커버리지, 문서화) |
| **Talent Pool CRM** | Lever, Gem | 탈락 후보자/관심 인재 장기 관리. 적합한 포지션 열리면 자동 알림 |
| **포트폴리오 페이지 생성** | Peerlist | Track B 완료 후 공개 포트폴리오 자동 생성 (후보자 브랜딩) |

### 🔌 고려할 API 및 통합
| 서비스 | 용도 | 우선순위 |
|--------|------|---------|
| **GitHub API** | 프로필 분석, 레포 분석, PR 이력 | 높음 |
| **LinkedIn API** (제한적) | 프로필 임포트 | 중간 |
| **Etherscan/Block Explorer API** | 온체인 활동 분석 | 높음 |
| **GitPOAP / Gitcoin Passport** | 기여 이력 검증 | 중간 |
| **OpenAI/Claude API** | Deliverable 자동 1차 분석 | 높음 |
| **SendGrid/Mailgun** | 아웃리치 이메일 자동화 | 높음 |
| **Slack/Discord API** | 리뷰어 알림, 후보자 커뮤니케이션 | 중간 |
| **Cal.com / Calendly API** | 인터뷰 스케줄링 | 낮음 |

### 💡 전략적 포지셔닝

> **기존 플랫폼의 한계**: Greenhouse, Lever 등은 "이력서 → 면접 → 채용"의 전통적 흐름에 최적화. 코드 평가 도구(HackerRank 등)는 알고리즘 퍼즐 중심으로 실제 업무 역량과 괴리.

> **Tokamak의 기회**: "실제 프로젝트 결과물 = 채용 평가" 모델은 Gitcoin의 바운티 시스템과 Greenhouse의 구조화 평가를 결합한 형태. 이 조합은 현재 시장에 존재하지 않음.

> **핵심 메시지**: "이력서가 아니라 코드로 증명하세요" — 블록체인/오픈소스 생태계에서 가장 설득력 있는 채용 방식
