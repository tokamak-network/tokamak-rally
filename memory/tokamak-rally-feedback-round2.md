# Tokamak Rally — Feedback Round 2 (2026-03-24)

## 1. Desert 배경색 통일
- 선인장 베이스의 밝은 모래색(황토+베이지)으로 전체 배경 색감 통일
- 현재 어두운 갈색 줄무늬 → 따뜻한 사막색으로

## 2. 황소(Cow) 개선
- 사이즈: 차량과 비슷한 크기로 키우기 (scaleMap 0.5 → 0.85+)
- 걷기 애니메이션: 2-3프레임 걷는 모션 추가
- 이동 방향: 머리가 향하는 방향으로 직진 이동 (좌우 sin wave → 직선 이동)

## 3. Canyon 존 개선
- 막대기(reflector) 파츠: 주변 배경색이 안 어울림 → 배경 제거 또는 파츠 삭제
- 환경과 안 어울리는 오브젝트 제거
- 배경 강화: 평지 → 계곡 사이의 길 컨셉 (Thrash Rally 코블스톤 협곡 참고)
- 도로 양쪽에 절벽 벽면, 바위기둥으로 깊이감

## 4. Sprint 존 개선
- 흰색 배너: Tokamak Network 로고(파란배너색) + 폰트(검은색) 추가
- 관중 뒤 건물: 듬성듬성 → 바둑판 배열로 빼곡하게 정렬
- 건물 옥상 "SPONSOR" → Tokamak Network 로고(글자 제외)로 교체
- 피니시 라인: 프레스센터, 포디움 시상대, 기자/카메라맨 배치

## 5. 존 전환 시스템 (Method A: Transition Zones)
5존 → 9존 (기존 5 + 전환 4)

### 전환 구간:
- **Desert → Canyon:** 사막 끝에서 바위/산맥 점진적 등장, 협곡으로 진입
- **Canyon → Riverbed:** 계곡 탈출 → 강/다리 → 물가로 전환
- **Riverbed → Mountain:** 초목이 침엽수로, 눈 패치 점진 증가
- **Mountain → Sprint:** 터널 구간 (어두운 오버레이 → 도시 조명으로 밝아짐)
