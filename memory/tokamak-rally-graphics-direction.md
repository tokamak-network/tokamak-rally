# Tokamak Rally — Graphics Direction (Confirmed 2026-03-24)

## Reference Games
- **Gameplay concept:** Thrash Rally (Neo Geo, 1991 by ADK) — top-down rally racing
- **Visual target:** Ultimate Racing 2D — pre-rendered 3D-style cars, realistic asphalt
- **NOT Kenney style** — too cartoony

## Confirmed Decisions (from Jaden)
1. **에셋 제작:** AI 이미지 생성 (DALL-E) — 코드 생성 방식 탈피
2. **Sprint 도로:** 아스팔트 (벽돌 아님)
3. **연속 배리어:** 모든 존에 추가
4. **차량+배경 동시 진행**
5. **퀄리티 우선** — 타겟: 블록체인 개발자 (고사양 기기)
