# Thrash Rally 레퍼런스 분석 — 트랙 디자인 원칙 (최종)

## 분석 소스
- 유튜브 영상: https://www.youtube.com/watch?v=KjMP7PxKixA (16:52)
- 플레이 영상 5개 — 총 243프레임 분석

## 핵심 트랙 디자인 원칙

### 1. 대각선 직선이 기본
- 수직 직선보다 30~45도 대각선이 더 빈번
- 대각선이 고속 주행의 기본, 수직 직선은 오히려 변주

### 2. 도로폭이 동적으로 변함
- 좁은 직선 → 코너에서 넓어짐 → 다시 좁아짐
- 넓은 코너 = 드리프트 유도, 좁은 직선 = 정밀 조향

### 3. 코너 예고 화살표
- 차량 전방, 도로 위 또는 바깥에 위치
- 빨간색 본체 + 흰/검정 테두리, 곡선형 (↱ ↰)
- 차량의 2~3배 크기로 크게
- 코너 1~2초 전에 표시

### 4. 코너 각도가 다양
- 45도, 90도, 180도 헤어핀 모두 등장
- 짧은/타이트 헤어핀 — 드리프트 극대화
- 넓은 헤어핀 — 고속 드리프트
- 진입각 ≠ 탈출각인 비대칭 코너도 존재

### 5. 배리어 유무/스타일이 구간마다 다름
- 배리어 있는 좁은 구간 ↔ 오픈 필드 교대
- 스타일: 대나무울타리, 돌담, 금속가드레일, curb, 나무울타리, 타이어벽
- 오픈 필드 = 코스 이탈 가능하지만 속도 페널티

### 6. 한 스테이지 내 다양한 환경 전환
- 아스팔트 → 비포장 → 돌길 → 모래길 → 나무다리
- 도로 유형에 따라 그립/속도/이펙트 변화

### 7. 점프대 / 나무다리
- 통나무/언덕 점프, 카메라 줌아웃 + 그림자 분리
- 나무다리: 물 위 극도로 좁은 폭, 추락 위험

### 8. 스타트/피니시 연출
- 열기구, 관중석, 스폰서 배너, 체커 깃발
- 관중이 도로 양쪽에 빽빽

### 9. 연출 요소
- 비행기/헬리콥터가 코스 위를 날아감
- 먼지 이펙트 (비포장)
- 충돌 폭발 이펙트

### 10. 타이어벽 / 코너 보호물
- 코너 바깥쪽에 타이어+배럴 계단식 배치
- 코너마다 보호물 다름 (타이어, 모래주머니, 배럴)

### 11. 배경 밀도 극히 높음
- 도로 양쪽 빈틈없이 나무/바위/동물/건물 채워짐
- 빈 배경 거의 노출 안 됨
- 배경 오브젝트가 도로와 일체감 형성

## 배리어 스타일 레퍼런스

| 환경 | 스타일 | 색상 |
|------|--------|------|
| 아스팔트/도시 | curb + 금속 가드레일 | 빨강/흰/은색 |
| 사바나/정글 | 대나무/나무 울타리 | 갈색/금색 |
| 산악/설산 | 돌담 또는 나무+눈 | 회색/갈색 |
| 해안 | 금속 가드레일 | 은색 |
| 오픈 필드 | 없음 | - |
| 코너 보호 | 타이어벽 + 배럴 | 다색 |
| 다리 | 나무 난간 | 갈색 |

- 크기: 차량의 약 2~3배
